# Assignment 1: Asset Pricing & Portfolio Construction
## RSM8341 Analytical Methods in Finance
### Team 10: Harsh Pinge, Gino Pak, Alex Griffith, Samir Hendawi, Wesley Thompson 

This project implements a full portfolio optimization pipeline in Python, 
covering return estimation, CAPM regression, and efficient frontier construction 
using both algebraic and quadratic programming methods. Using daily S&P 500 
constituent prices, market index data, and 10-year Treasury rates spanning 
over five decades, we apply the MAXSER framework — including Lasso regression 
and Sharpe ratio estimation — to identify optimal portfolio weights. The analysis 
benchmarks the plug-in solution against the MAXSER approach, highlighting 
practical differences in portfolio construction under real market conditions.
